# Infrastructure

## Purpose
Provides local development and deployment scaffolding for TutorBot Campus dependencies and runtime topology.

## Responsibilities
- Start local middleware and databases
- Model a fuller end-to-end stack for local integration testing
- Hold Kubernetes deployment notes and future manifests

## Key Files
| File | Description |
|------|-------------|
| docker-compose.dev.yml | Lightweight dependency stack for day-to-day development |
| docker-compose.full.yml | Full local stack including all application services |
| k8s/README.md | Placeholder for Kubernetes manifests and deployment notes |
