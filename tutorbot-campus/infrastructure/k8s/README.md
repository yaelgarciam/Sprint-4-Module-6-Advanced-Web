# Kubernetes

## Purpose
Placeholder directory for Kubernetes manifests, Helm charts, and deployment overlays.

## Responsibilities
- Define environment-specific manifests for each service
- Hold ingress, secret, and configuration examples
- Capture scaling and observability deployment notes

## Planned Assets
- Namespaces and network policies
- Deployments and Services for each backend component
- ConfigMaps and Secrets for environment settings
- HorizontalPodAutoscaler definitions for tutoring agents
