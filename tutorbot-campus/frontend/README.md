# Frontend

## Purpose
Hosts the React + Vite web application for students, tutors, and university operators.

## Responsibilities
- Render student tutoring workflows and dashboards
- Integrate with the API gateway and real-time notification endpoints
- Organize reusable UI components, hooks, pages, and service clients

## Tech Stack
| Technology | Usage |
|------------|-------|
| React | UI composition |
| Vite | Frontend tooling and bundling |
| TypeScript-ready structure | Safe growth path for the web app |
| WebSocket / REST clients | Real-time and API communication |
